🎮 Game Engine

State Machine

Bet Placement

  • Rate limit: 5/sec, 60/min per user
  • Lock game_rounds FOR UPDATE
  • Lock user_balances FOR UPDATE
  • Debit via LedgerService (idempotent)
  • ≥2 players → transition to active
  • 30-second countdown starts

Winner Selection

  • SHA-256 combined hash (server + client seeds)
  • Binary search on cumulative weights
  • Payout: pot − 2% commission − 1% referral
  • Strict lock ordering (deadlock prevention)
  • 3x retry on deadlock (50/100/150ms)
  • Immutable snapshot stored

Provably Fair System

Every round is independently verifiable. Server seed (64-char hex) is committed before betting via SHA-256 hash, revealed after finish.

Game Worker

  • Redis Streams consumer (XREADGROUP BLOCK 5000)
  • 2 replicas for high availability
  • Heartbeat every 10s (SETEX worker:{name}:heartbeat 30)
  • Graceful shutdown on SIGTERM
  • Dead worker recovery via XCLAIM
  • Publishes game:finished to Redis Pub/Sub

Safety Guarantees

Race Condition Prevention

  • SELECT ... FOR UPDATE on all financial ops
  • Strict lock ordering (deadlock prevention)
  • Balance check before debit (atomic)
  • Idempotent via unique constraints

Invariant Checks (Reconciliation)

  • No money created from nothing
  • No money lost (sum of all entries = 0)
  • Every balance traceable to ledger
  • Runs every 5 minutes via cron

Crypto Payments (NOWPayments)

Deposits

  • Invoice creation via API
  • Webhook status tracking
  • Overpayment protection
  • Rate limit: 5/hour

Withdrawals

  • Min $5, max $10k/day
  • >$500 requires admin approval
  • Atomic: debit → payout → refund on fail
  • HMAC-SHA512 webhook security

🛡️ Security & Anti-Fraud

🔒 Data Protection

  • Session-based auth with guards
  • JWT tokens with refresh rotation
  • HMAC-SHA512 webhook validation
  • hash_equals() timing-attack safe
  • Row-level locking on all financial ops
  • Disposable email blocking
  • Nicknames instead of emails (privacy)

🕵️ Fraud Detection

  • Win streak auto-flagging (≥10 wins/24h)
  • High velocity detection (≥50 bets/5min)
  • IP correlation (shared registration IPs)
  • Large withdrawal alerts (>$1000/7d)
  • Device fingerprinting (canvas hash)
  • Admin Activity Monitor dashboard
